# Annida

Annida Foundation website (Vite + Preact).

## Development

From project root:

```
npm run dev
```

This runs the client dev server from `Client_Annida` using Vite and serves at http://localhost:5173 by default.

Or run locally in the client folder:

```
cd Client_Annida
npm install
npm run dev
```

## Build

```
cd Client_Annida
npm run build
```
